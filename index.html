Bard is an open-source session replay tool for small eCommerce businesses. Users can record, store, replay and analyze user sessions, allowing them to debug and optimize their user interface and conversion funnels.

Built using Rrweb, Bard combines industry-leading session recording technologies and an intuitive user interface ensure that the recorded data is collected, stored, and replayed efficently.

With Bard, the user has the option to view session individully using filter or as part of a conversion funnel. Bard correlates each session with steps in a conversion funnel that lets you gain visibility into why users are not converting.
